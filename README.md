# ファッション推薦システム (Fashion-IGA-GenAI-System)

感性工学と生成AIを組み合わせた、インタラクティブ遺伝的アルゴリズム（IGA）によるファッション提案システムです。

## プロジェクト概要
ユーザーのファッションに対する感性（好み）を学習し、一人ひとりに最適なスタイルを提案します。
本研究では、遺伝的アルゴリズムによる進化プロセスと、Stability AIによる高精細な画像生成を統合しています。

## プロジェクト構成
├── src/main/java/com/example/stabilityai/

│   ├── Main.java                # エントリポイント・GUI制御

│   ├── FashionGene.java         # 遺伝子モデル・プロンプト生成

│   ├── GenePool.java            # GAアルゴリズム（交叉・突然変異）

│   └── StabilityAIImageGenerator.java # API通信部

├── pom.xml                      # Maven設定

└── .gitignore                   # セキュリティ設定

## 使用技術
- **言語**: Java 23 (最新のプレビュー機能や構文を活用)
- **ビルドツール**: Maven
- **外部API**: Stability AI (Stable Diffusion XL 1.0)
- **アルゴリズム**: 対話型遺伝的アルゴリズム (IGA)
    - 候補解数 8
    - 最大世代数 10
    - 遺伝子長 8 
    - エリート選択、ルーレット選択
    - エリート数 1
    - 一様交叉(0.9)
    - 突然変異(0.1)

## 主な機能
- **動的プロンプト生成**: 遺伝子情報からAI用のプロンプトを自動生成
- **セキュリティ**: APIキーや環境依存情報を .env ファイルで安全に管理

## セットアップ (Setup)

本システムを実行するための手順です。

### 1. APIキーの設定
1. [Stability AI Platform](https://platform.stability.ai/) でアカウントを作成し、APIキーを取得します。
2. プロジェクトのルートディレクトリ（`pom.xml` と同じ場所）に `.env` という名前のファイルを作成します。
3. 作成したファイル内に以下の内容を記述し、保存してください。
   ```text
   STABILITY_AI_API_KEY=ここにあなたのAPIキーを記述
※ .gitignore により、このファイルは公開されないよう設定されています。

### 2. ライブラリのインストール
Mavenを使用して、必要なライブラリをインストールします。

Bash
mvn install
### 3. アプリケーションの実行
以下のコマンドでメインクラスを起動します。

Bash
mvn exec:java -Dexec.mainClass="com.example.stabilityai.Main"

### 技術的な工夫
- **再現性の確保**: 同じ遺伝子情報（プロンプト）からは常に同じ画像が生成されるよう、プロンプトのハッシュ値をAPIのシード値として利用しています。
- **UI/UX**: Java Swingを用いた直感的なインターフェースにより、研究者が実験データを収集しやすい設計にしています。
